

# The Fastest Low Light Enhancement — Android App (Graduation Project)

> This repository contains the Android application developed as part of the graduation project “The Fastest Low Light Enhancement.”
The project seeks to achieve real-time video enhancement in low-light environments by deploying a lightweight Illumination-Adaptive Transformer (IAT) model on mobile devices. The application implements real-time correction of camera input, demonstrating the practical integration of the model within an Android system.
